# vite-press-demo 项目源码

## 运行项目

### 安装 pnpm（如果未安装）

```shell
npm install -g pnpm
```

### 安装依赖

```shell
pnpm install
```

### 项目构建

```shell
pnpm run docs:build
```

### 本地运行

```shell
pnpm run docs:dev
```

打开：[http://localhost:5174/vite-press-demo/](http://localhost:5174/vite-press-demo/)
